# 🛡️ AdGuard Home 规则库
---
## 🤔 简介
融合多个Adguard规则，最终筛选出适用于Adguard Home使用的DNS规则  
- 全部规则：仅去重  
- OAdH规则(OAdH_ALL)：去重后仅DNS规则 **(推荐)**
- OAdH去冲突规则(OAdH_NCR)：去除黑白名单同时存在的情况 **(实验性)**
